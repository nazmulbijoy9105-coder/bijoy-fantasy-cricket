Quick Start

One Request Away

example — get match predictions 
# Free: get top scorers in BPL 2025
curl https://bpl-cricket-api.onrender.com/bpl/matches/season/2025/top-scorers

# Response
{
  "season": 2025,
  "top_scorers": [
    { "name": "Shakib Al Hasan", "runs": 612, "matches": 12, "avg_sr": 148.6 },
    { "name": "Tamim Iqbal",     "runs": 580, "matches": 11, "avg_sr": 135.2 }
  ]
}

# Paid: AI prediction for match 42
curl -H "Authorization: Bearer bpl_YOUR_API_KEY" \
     https://bpl-cricket-api.onrender.com/bpl/premium/predictions/42

# Response
{
  "predicted_winner": { "name": "Fortune Barishal", "confidence": 0.78 },
  "predicted_top_scorer": { "name": "Shakib Al Hasan", "runs": 612 },
  "model": "statistical_v1"
}

Deployment

Live in 5 Minutes

Free hosting on Render.com. No credit card required. Singapore region for low latency.

01
Clone & push to GitHub
Download the project zip, unzip it, init a git repo and push.
git init && git add . && git commit -m "init" && gh repo create bpl-api --public --push
02
Connect Render.com
New Web Service → connect your GitHub repo. Render auto-reads render.yaml.
render.yaml handles: build, start, env vars, health check
03
Add payment keys (optional)
Add BCASH_API_KEY and NAGAD_MERCHANT_KEY in Render dashboard environment settings.
Environment → Add variables from .env.example
04
Hit your live API
Render gives you a public URL. Open /docs for the full Swagger UI.
https://bpl-cricket-api.onrender.com/docs
05
Scrape real data
Run the scrapers locally to populate your DB with real BPL numeric stats.
python -m scraper.scrape_bpl_season --all
